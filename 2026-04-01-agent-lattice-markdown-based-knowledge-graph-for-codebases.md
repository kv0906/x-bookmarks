# Agent Lattice: Markdown-based Knowledge Graph for Codebases

## Summary
This bookmark is for Agent Lattice (lat.md), a workflow and toolset that creates a knowledge graph for a codebase, entirely written in Markdown. It's designed to enhance how both humans and AI agents understand and interact with code, aiming to improve knowledge retention, code review, and automated agent workflows.

## Key Insights
- Agent Lattice uses Markdown files to represent a codebase's knowledge graph, making it human-readable and version-controllable.
- It comes with a CLI (`lat init`, `lat check`, `lat search`, `lat section`) to manage and navigate the graph, enforcing referential consistency.
- The system integrates with pre-commit hooks and GitHub bots, allowing agents to automatically update and maintain the knowledge graph.
- Agent Lattice helps retain context and reasoning behind coding decisions, which is often lost in traditional workflows, by having agents capture knowledge into the graph.
- Test specifications can be defined within the Markdown graph and linked to actual code via comments, with `lat check` ensuring coverage and consistency.
