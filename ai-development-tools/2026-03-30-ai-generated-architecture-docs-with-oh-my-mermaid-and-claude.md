# AI-Generated Architecture Docs with Oh-My-Mermaid and Claude Code

## Summary
This bookmarked tweet highlights 'oh-my-mermaid', an open-source tool that leverages AI (specifically Claude Code) to automatically generate and maintain architecture documentation from complex codebases. It addresses the challenge of code becoming a 'black box' by creating clear, navigable architecture diagrams and structured documentation that is synchronized with the code. The linked GitHub repository details how the tool works, generating hierarchical documentation with Mermaid diagrams and descriptive text, making it a valuable asset for developers and architects.

## Key Insights
- Oh-my-mermaid generates architecture documentation by having an AI (like Claude Code) analyze a codebase and produce 'perspectives' including Mermaid diagrams and documentation fields.
- It creates a structured `.omm/` directory containing `description.md`, `diagram.mmd` (Mermaid), and other contextual files for each architectural component, supporting nested elements.
- The tool transforms architecture documentation into a 'living artifact' that is versioned, continuously updated, and actively used throughout the system's lifecycle, addressing common issues with outdated diagrams.
- Mermaid is a text-based diagramming language that is easily parsable by AI, enabling automated generation of various diagram types (flowcharts, sequence diagrams, C4 models) that are also human-readable and version-controllable.
