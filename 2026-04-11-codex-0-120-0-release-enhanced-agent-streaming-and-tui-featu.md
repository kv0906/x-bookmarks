# Codex 0.120.0 Release: Enhanced Agent Streaming and TUI Features

## Summary
This bookmark highlights the release of Codex 0.120.0, introducing significant improvements for developers working with AI agents and terminal user interfaces. The update focuses on more transparent agent execution with live progress streaming, clearer TUI organization of hooks, and better support for structured tool outputs. The linked GitHub release details new features like Realtime V2's ability to stream background agent progress, improved TUI readability for hook activity, and enhanced Model Context Protocol (MCP) support for typed tool results and custom servers, alongside various bug fixes and performance optimizations.

## Key Insights
- Codex agents now provide live progress streaming, allowing users to see agent activity as it happens and queue follow-up responses.
- The Terminal User Interface (TUI) has been refined to separate live and completed hooks, making agent execution easier to monitor and understand.
- MCP tools now support `outputSchema`, enabling more precise typing for structured results from tool declarations, which improves reliability for AI-powered clients.
- The release includes broader support for MCP Apps and custom MCP servers, providing richer capabilities for resource management, tool-call metadata, and plugin cache refreshes.
