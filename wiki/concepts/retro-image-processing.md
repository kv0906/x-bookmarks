# Retro Image Processing

A cluster of tools for achieving pixelated, dithered, and retro visual aesthetics — all bookmarked on the same day (2026-03-27), suggesting a moment of active interest in this space. The tools range from shader libraries to dedicated dithering software to real-time WebGPU effects.

## Key Points
- **Paper Shaders** — zero-dependency WebGL library with 30+ effects (dithering, halftone), usable via Paper.design platform or npm for React/vanilla JS
- **Dither Boy** — dedicated dithering software with 63 unique algorithms, supporting full color, video, and animation with stackable effects pipeline and timeline editor
- **Grainrad** — free browser-based tool using WebGPU for real-time ASCII art, dithering, and retro effects at 60fps
- Common thread: all three are browser-accessible, no install required, and designed for creative/artistic use rather than production image processing
- Technology spectrum: WebGL (Paper Shaders) → native app (Dither Boy) → WebGPU (Grainrad)
