# Scientific Research Agents

AI agents purpose-built for research workflows — from literature review to experiment replication to automated iterative improvement. These go beyond chatbot-style Q&A to execute multi-step research processes autonomously.

## Key Points
- **Feynman** is an open-source agent that generates cited meta-analyses, replicates experiments on RunPod, audits claims against code, and simulates peer review — MIT licensed
- **Autoresearch** (from Karpathy) automates iterative improvement of prompts/skills using objective scoring metrics — improved a Claude Code skill from 41% to 92% in 4 rounds overnight
- Pattern: research agents work best when they have clear evaluation criteria and can loop through improve→test→score cycles
