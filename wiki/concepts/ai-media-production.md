# AI Media Production

AI tools transforming how media is created — from course content to music production to video editing. The common thread is AI handling the tedious production work while humans focus on creative direction.

## Key Points
- **ButterCut** uses Claude Code + WhisperX to analyze footage, transcribe audio, and generate rough cuts for Final Cut Pro, Premiere, and DaVinci Resolve — open source
- **AI vocalists** (Voxyz.ai) streamline vocal production — alignment, tuning, and synthesis replace physical studio sessions with cloud-based platforms
- **AI course creation** covers the full pipeline: audience research, content generation, platform setup, and launch campaigns using AI tools
- Pattern: AI excels at the time-consuming middle steps (transcription, rough cuts, vocal alignment) while humans own the creative vision and final polish
