# OpenAI Codex Launches Plugins for Enhanced Developer Workflows

## Summary
OpenAI has rolled out a new plugin system for its Codex coding agent, designed to integrate seamlessly with popular developer tools like Slack, Figma, Notion, and Gmail. This enhancement allows users to package and distribute reusable Codex workflows, including skills and app integrations, making the development process more efficient and collaborative.

## Key Insights
- Codex plugins are installable bundles that package reusable workflows, skills (prompts for specific tasks), and optional app integrations, enabling consistent configurations across teams and projects.
- The plugin system supports integrations with widely used platforms such as Slack, Figma, Notion, and Gmail, allowing developers to trigger coding tasks, translate designs to code, and manage workflows directly within these applications.
- Plugins leverage the Model Context Protocol (MCP), an open standard that acts as a universal adapter for AI, providing real-time context from various sources like design files or databases to the AI model.
- While 'Codex' initially referred to specific models deprecated in March 2023, the term now encompasses OpenAI's broader suite of developer tools, powered by newer GPT models, that facilitate code generation, understanding, review, and automation.
